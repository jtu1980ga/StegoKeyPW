# StegoKeyPW
Hides an encrypted secret message in an image file that is password protected.
Prompts ask to Encode or Decode.
ENCODE
When you select encode, It asks for a password,  that is used to password protect the encyption key and the hidden message.
  It asks,  which image file you would like to save the Encrypted Secret messag1e into using stegonography.
  It then prompts for  you to type your hidden secret message.
  Once you enter your Secret Message, it will ask you to name the new image file which contains the hidden message.
 DECODE
 When you select Decode, It asks you for the password to decocde the Hidden message,  
 Then it asks you which image file you are trying to decode,  If you entered the correct password that was used to do the encoding, your secret hidden message will be revealed on the screen.  All Users Prompts are GUI prompts.
